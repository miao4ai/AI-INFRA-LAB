# AI-INFRA-LAB

> **Building modern AI infrastructure from distributed systems to world models.**

AI-INFRA-LAB is a long-term engineering and research roadmap focused on the systems behind modern artificial intelligence.

Rather than collecting isolated demos, this repository is organized as a cohesive ecosystem that spans distributed computing, AI infrastructure, large-scale inference, and next-generation foundation models.

The goal is to understand how modern AI systems are built—from CPU scheduling and distributed computation to GPU inference, multimodal pipelines, and embodied intelligence.

---

# Vision

Modern AI is no longer just about training large language models.

It is about building the entire stack:

```
Computer Architecture
        ↓
Operating Systems
        ↓
Distributed Computing
        ↓
Cloud Infrastructure
        ↓
AI Infrastructure
        ↓
Foundation Models
        ↓
World Models
        ↓
Embodied AI
```

AI-INFRA-LAB follows this philosophy by learning every layer through hands-on implementation.

# Projects

## PiHPC-Lab

A Raspberry Pi based HPC laboratory for learning distributed systems from the ground up.

Topics include:

* MPI
* OpenMP
* Spark
* Hadoop
* Kubernetes
* Ray
* Distributed Algorithms
* Cluster Deployment
* Performance Benchmarking

---

## SentinelAI

A production-style multimodal AI platform.

Features include:

* Distributed video processing
* FFmpeg pipelines
* Spark ETL
* Airflow orchestration
* Embedding generation
* Vector Search
* GPU inference
* Kubernetes deployment

This project focuses on production AI infrastructure.

---

## InferMesh

A local-first distributed AI inference framework.

Goals:

* AI inference orchestration
* GPU scheduling
* Distributed workers
* Load balancing
* Model serving
* Edge inference
* Cloud-native deployment

Inspired by the idea:

> Everything infers locally. Infrastructure orchestrates globally.

---

## WorldLab

A research playground built with JAX.

Unlike production-oriented repositories, WorldLab focuses on understanding modern AI models through implementation from scratch.

Topics include:

* JAX fundamentals
* Transformer implementation
* Vision Transformers
* MAE
* VideoMAE
* JEPA
* V-JEPA
* Dreamer
* World Models
* Reinforcement Learning
* Embodied AI
* OpenVLA
